🚀 إنشاء Worker (5 دقائق، مجاناً)

قنوات beIN Arabic محمية بفحص Referer على السيرفر. لتشغيلها نحتاج وسيط بسيط على Cloudflare Workers (100 ألف طلب يومياً مجاناً).

  1. اذهب إلى workers.cloudflare.com وأنشئ حساب مجاني
  2. اضغط Create → Hello World → سمّه tifotv-bein
  3. اضغط Edit Code → احذف كل الكود الموجود
  4. الصق محتوى ملف worker.js المرفق معك بالكامل
  5. اضغط Save and Deploy
  6. انسخ رابط الـ worker الظاهر (مثل https://tifotv-bein.user.workers.dev) وضعه أعلاه

✓ بعدها ستعمل كل الـ 37 قناة beIN + SSC + Dubai تلقائياً!
