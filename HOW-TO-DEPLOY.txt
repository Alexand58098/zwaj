  ⚽ TifoTV — تعليمات النشر على Netlify (مهم!)
═══════════════════════════════════════════════════════════════

⚠️ المشكلة السابقة:
   لو رفعت الـ ZIP أو سحبت المجلد الأب → صفحة 404
   لأن Netlify يبحث عن index.html في الجذر مباشرة

✅ الحل الصحيح:

  الطريقة 1 — drag & drop (الأسهل):
  ──────────────────────────────────
  1. افتح: https://app.netlify.com/drop
  2. افتح هذا المجلد (tifotv-flat) في مستكشف الملفات
  3. حدّد كل الملفات داخله: Ctrl+A
  4. اسحب الملفات (وليس المجلد!) إلى صفحة Netlify
     يجب أن يظهر "index.html" في قائمة الرفع مباشرة في الجذر
  5. انتظر 30-60 ثانية
  6. سيظهر رابط .netlify.app ← افتحه

  ⚠️ لو ظهرت صفحة 404:
     - تحقق من Netlify Dashboard → Deploys
     - اضغط على آخر deploy
     - تأكد أن "index.html" موجود في الـ "Deploy file browser"
     - إذا كان داخل مجلد فرعي → سحبت خطأ، أعد الرفع

  الطريقة 2 — رفع بـ ZIP:
  ───────────────────────
  1. اضغط المحتوى (وليس المجلد) في ZIP:
     - افتح المجلد
     - حدّد كل الملفات (Ctrl+A)
     - Right-click → Compress → اسم: tifotv-zip
  2. ارفع الـ ZIP الناتج إلى Netlify Drop

═══════════════════════════════════════════════════════════════

🔍 كيف تتأكد أن النشر صحيح؟

  1. افتح: https://YOUR-SITE.netlify.app
     ← يجب أن ترى الموقع (ليس 404)

  2. افتح: https://YOUR-SITE.netlify.app/.netlify/functions/bein
     ← يجب أن ترى JSON يقول: { "status": "ok", "name": "TifoTV..." }

  3. افتح: https://YOUR-SITE.netlify.app/.netlify/functions/bein?id=91&json=1
     ← يجب أن ترى JSON يحتوي على رابط m3u8

  إذا كل هذا صحيح، الموقع يعمل 100% ✓

📁 محتوى هذا المجلد (يجب أن يكون كل هذا في جذر الموقع):

  index.html          ← الموقع الرئيسي (لازم في الجذر!)
  netlify.toml        ← إعدادات Netlify
  _redirects          ← redirects بديلة
  package.json        ← لـ Netlify Functions
  netlify/
    functions/

  worker.js           ← بديل Cloudflare
